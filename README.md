# Student Organization Management System (SOMS)

## Short Description
The Student Organization Management System is a web-based application built with Django. It provides a centralized platform for managing student organizations. The system streamlines administrative tasks by offering features such as:

## List of Features
1. User Authentication: Secure login and role-based access for students, officers, and administrators.
2. Organization Management: Create, update, and track student organizations with member rosters.
